## Run

Usage: ```docker run [OPTIONS] IMAGE [COMMAND] [ARG...]```

```-d``` 在后台运行容器并显示容器标识  
```-p list``` 将容器的端口发布到主机  
```-v list``` 绑定一个卷(文件或目录)  
```-link list``` 添加一个链接到其它容器
